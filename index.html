All BedWars Commands

Gamemode Command

This command allows Hosts/Co-Hosts to change the gamemode of any player in the match.

/gm {1/0}

/gm {1/0} @all

/gm {1/0} {Username}

/gm {1/0} @team:{Team Color}

Edit Commands

This command allows Hosts/Co-Hosts to edit the map using EasyEdit.

//edit flat {Block Name} {Size : Number}

//cut

//set {Block Name}

//undo

//redo

//pos1

//pos2

//flat {Block Name} {Size : Number}

//replace {Block to Replace} {Block to Set}

Generator Commands

These commands allow Hosts/Co-Hosts to create/delete/manage any generator placed on the map.

/gen list

/gen remove {Generator ID}

/gen move {Generator ID}

/gen create global {Item Name} {Drop Delay} {Stack Limit}

Shop Commands

These commands allow Hosts/Co-Hoests to create/delete any shop merhcant placed on the map.

/shop list

/shop remove {Shop ID}

/shop create {item/upgrade}

SetTeamSpawn Command

This command allows Hosts/Co-Hosts to set the spawn point for a specific team.

/setteamspawn {Team Color}

Spawn Commands

These commands allow Hosts/Co-Hosts to give themselves any item in the game.

/spawn {Item Name} {Quantity}

/i {Item Name} {Quantity}

SetGeneratorMultiplier Command

This command allows Hosts/Co-Hosts to change the delay between item drops on all game generators.

/setgeneratormultiplier {Delay}

Teleport Command

This command allows Hosts/Co-Hosts to teleport Player1 to Player2.

/tp {Player1/Player2} {Player2 : Optional}

Give Command

This command allows Hosts/Co-Hosts to give any item to any player in the match.

/give {Username} {Item Name} {Quantity}

SetTeam Command

This command allows Hosts/Co-Hosts to set the team of any player, players on a team, or all players.

/setteam {Username} {Team Color}

/setteam @a {Team Color}

/setteam @team:{Team Color} {Team Color}

Enchant Command

This command allows Hosts/Co-Hosts to give themselves any enchant in the game.

/enchant {Enchant Name}_{Level : Number}

SetHealth Command

This command allows Hosts/Co-Hosts to set the health of any player, team of players, or all players.

/sethealth {Username} {Health}

/sethealth @a {Health}

/sethealth @team:{Team Color} {Health}

Disaster Command

This command allows Hosts/Co-Hosts to trigger any disaster in the game.

/disaster {meteors/void_rise/toxic_rain/tornado}

CoHost Command

This command allows Hosts/Co-Hosts to give the CoHost permission to all players.

/cohost all

SetHealthRegen Command

This command allows Hosts/Co-Hosts to set the health regeneration amount for any player.

/sethealthregen {Username} {Regen Amount}

TpBed Command

This command allows Hosts/Co-Hosts to teleport to any team's bed.

/tpbed {Team Color}

SetDeathmatchTime Command

This command alllows Hosts/Co-Hosts to set the timer for the server bed break.

/deathmatchtime {Time : Seconds}

Place Teleport Commands

These commands allow players to teleport to the BedWars lobby, Islands, or Prophunt. All of which are created by Easy Games.

/bedwars

/lobby

/islands

/prophunt

Disabled

BlacklistItem Command

This command allows Hosts/Co-Hosts to blacklist any item in the game from being purchased from the ItemShop merhcant.

/blacklistitem {Item Name}

Airdrop Commands

These commands allow Hosts/Co-Hosts to trigger an airdrop of a Juggernaut crate or Purple Lucky Blocks.

/airdrop {juggernaut/luckyblock}

/luckyblockairdrop

TpAll Command

This command allows Hosts/Co-Hosts to teleport all players to themself.

/tpall

Kick Command

This command allows Hosts/Co-Hosts to kick any player in a match.

/kick {Username}

Announce Command

This command allows Hosts/Co-Hosts to make an announcement notification for all players.

/announce {Message}

DisableKit Command

This command allows Hosts/Co-Hosts to disable any kit in the game.

/disablekit {Kit Name/all}

ToggleTeamOverflow Command

This command allows Hosts/Co-Hosts to allow team player count overflowing.

/toggleteamoverflow

ToggleSpawn Command

This command allows Hosts/Co-Hosts to toggle the spawning of any entity in the game.

/togglespawn {Entity Name}

Bed Commands

These commands currently have no function but have a confirmed release soon!

/bed list

/bed remove {Bed ID}

/bed create {Team Color}

AntiCheat Manage Command

This command allows Hosts/Co-Hosts to enable/disable the game's AntiCheat.

/setac {enabled/disabled}

Cube Command

This command allows spawns a giant cube made of White Wool, Oak Planks, and Stone Brick.

/cube
